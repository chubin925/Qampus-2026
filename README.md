# Qampus
학교 생활 중 생기는 궁금증을 질문하고 답변할 수 있는 익명 질의응답 게시판입니다.

## ✨ 주요 기능

- 게시글 CRUD
- 카테고리별 게시글 조회
- 추천순 / 최신순 정렬
- 댓글 / 대댓글 CRUD
- 게시글 좋아요
- 댓글 좋아요
- 대댓글 좋아요
- 게시글 스크랩
- 이미지 업로드

## 🛠 Tech Stack

| 분야 | 기술 |
|------|------|
| 📋 Planning & Design | Notion, Figma |
| 💻 Frontend | HTML5, CSS3, JavaScript |
| ⚙️ Backend | Python 3.13, Django 6.0 |
| 🗄️ Database | SQLite3 |
| 🤝 Collaboration | Git, GitHub |
